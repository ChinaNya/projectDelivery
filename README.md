# Информационная система для оптимизации маршрутов доставки в малых городах

Данный проект представляет собой дипломную работу студента специальности 09.02.07 Информационные системы и программирование. Разработанное веб-приложение предназначено для организации доставки товаров в условиях малого города и объединяет клиентов, курьеров и местные магазины.

---

## Описание

Система позволяет:
- **Клиентам** – просматривать список товаров, формировать заказ, указывать адрес доставки, отслеживать статус выполнения и оценивать работу курьера.
- **Курьерам** – выбирать график работы, просматривать доступные заказы, получать оптимальный маршрут до магазина и клиента, отмечать этапы доставки, видеть свою статистику и рейтинг.

Все данные сохраняются в браузере с помощью **LocalStorage**, что позволяет обходиться без серверной части и разворачивать приложение на любом хостинге статических сайтов.

---

## Функциональные возможности

### Для клиента
- Регистрация / авторизация.
- Просмотр товаров (хлеб, молоко, яблоки, пицца, суши и др.).
- Добавление товаров в корзину.
- Оформление заказа с указанием адреса и комментария.
- Отслеживание заказа:
  - номер заказа,
  - статус (поиск курьера / в пути),
  - таймер с момента создания,
  - контактный телефон курьера,
  - возможность отмены (с правилами возврата).
- После доставки – оценка курьера (звёзды), отзыв, чаевые.

### Для курьера
- Регистрация / авторизация.
- Выбор рабочего графика (доступные слоты зависят от текущего времени).
- Просмотр списка свободных заказов.
- Принятие заказа:
  - автоматическое определение ближайшего магазина,
  - построение маршрута на карте (сначала до магазина, затем до клиента),
  - поэтапные кнопки: «Прибыл к пункту» → «Заказ взят» → «Заказ доставлен».
- Отображение расстояния до цели.
- Завершение заказа – переход к списку доступных.

### Общее
- Личный кабинет с историей заказов.
- Для курьеров – отображение рейтинга.
- Кнопка выхода из профиля.
- Встроенная справка для курьеров.

## Используемые технологии

- **HTML5, CSS3** – разметка и стилизация.
- **JavaScript (ES6+)** – вся логика приложения.
- **Bootstrap** – адаптивный интерфейс.
- **Leaflet.js** – отображение карт (OpenStreetMap).
- **LocalStorage** – хранение данных пользователей, заказов и сессий.
- **Geolocation API** – определение местоположения курьера.
